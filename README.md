# VR_Telemedicine
협동로봇과 VR기술로 원격의료 시스템 개발


## 상태
* **진행** : 4 단계에서 무기한 중지
    * 사유 : 환경변화
* **기간** : 2022-08-17 ~ 2023-03-01
* **인원** : 1명


## 개요
* 1 단계 : 협동로봇 제어 확립
* 2 단계 : 협동로봇 정밀 제어
* 3 단계 : 협동로봇 제어 응용
* 4 단계 : VR콘솔 데이터 추출
* 5 단계 : VR콘솔과 협동로봇 연동
* (궁극적인 방향) 6 단계 : 원격제어 데이터로 AI 모델 생성


## 개발의의
* 원격의료 
* 협동로봇의 고등응용으로 실력 향상
